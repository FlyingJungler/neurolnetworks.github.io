How it works

Input

Data representation

The input layer represents the raw data that the neural network processes. Each input node corresponds to a feature or variable in the data, and their values are directly mapped to the next layer without any computation.

Dimensional

The number of nodes in the input layer is determined by the dimensional of the input data. It reflects the number of features or variables in the input dataset, such as pixels in an image or words in a text sequence.

Role in Training

During training, the input layer's primary role is to receive and pass the data to the hidden layers. It does not perform computations on the data, leaving that task to the subsequent layers that make up the core processing of the neural network.

Hidden

Data Transformation

Hidden layers are where the core processing of the neural network occurs. Each node in a hidden layer takes the weighted sum of its inputs and applies an activation function. This transformation allows the network to capture complex relationships within the data.

Multi-layer Architecture:

Neural networks often consist of multiple hidden layers, creating a deep architecture. Deep learning, which involves deep neural networks, has proven effective in learning intricate, hierarchical features in data.

Hierarchical Features

The hidden layers learn to extract hierarchical features from the input data. Lower layers might capture simple features like edges in an image, while higher layers can recognize more complex patterns like shapes or objects.

Non-Linearity

The activation function applied in each node of the hidden layers introduces non-linearity, enabling neural networks to model and approximate non-linear relationships in the data. Popular activation functions include Rectified Linear Unit (ReLU), sigmoid, and hyperbolic tangent (tanh).
